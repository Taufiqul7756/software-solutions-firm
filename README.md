## Getting Started

### Prerequisites

- Node.js: Make sure you have Node.js installed on your system.

### Installation

1. Clone the repository from GitHub.

   ```shell
   git clone <repository_url>
   ```

   ```shell
   cd <project_directory>
   ```

   ```shell
   npm install
   ```

   ```shell
   npm run dev
   ```
